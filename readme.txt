== Description ==

Quick View and Buy Now plugin makes the buying process easy in your store to increase conversion and encorage clients buying from your website by adding Quick View popups and Buy Now buttons.

The plugin offers many features in both Free and Pro Versions.


* Change The redirect-to-cart option to redirect to checkout.


* Change "Add To Cart" Buttons Text.


* Change "View Cart" and "Continue Shopping" links in added to cart notice with "Checkout" link.


* Add "Quick View" button to products in shop and archive pages [ simple - variable - grouped - external ]


* Show the product details in a popup with the ability to choose Quantity and either "Add to Cart" or "Buy Now"


* Add "Buy Now" button in single product pages for product types [ simple - variable - grouped ]


* enable / disable the "Quick View" button by product type.


* Customize the "Quick View" Button Title.


* Customize the "Quick View" Button position.


* enable / disable the "Buy Now" Button by product type.


* Customize the "Buy Now" Button Title.


* Customize the "Buy Now" Button position.


* Choose to show or hide additional sections in popup product [ Data tabs section - Upsells products section - related products seciton ].


* Select the redirect page after "Buy Now" between cart page or checkout page.


* Add custom styles for the "Quick View" - "Buy Now" Buttons and the Quick View popup.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Settings page in `WooCommerce/Settings/Quick View and Buy Now` tab
